# NumberToLetter

This package is responsible for transforming numbers into text statements.

## Installation

Install NumberToLetter with npm

```bash
  npm install numlett
```

```javascript
import numlett from "numlett";

numlett(300);
// TRESCIENTOS
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
